# ASIN Stock vs Sales Regional Dashboard

## Project Overview
This mini project analyzes product sales regions and compares them with current stock availability using geographic dashboards.

## Objective
- Analyze where the product was sold
- Compare current stock distribution
- Identify demand vs inventory coverage by region

## Tools Used
- Microsoft Excel
- Pivot Tables
- Map Charts
- Inventory Analytics

## Dashboard Features
- Sales Region Mapping
- Stock Region Mapping
- State-wise Inventory Analysis
- Demand vs Stock Comparison
